# Python CLI To-Do List

## Project Overview

This is a simple command-line interface (CLI) to-do list application built with Python. It allows you to add, view, and delete tasks directly from your terminal, including the integrated terminal in Visual Studio Code.

## Features

- **Add Tasks**: Add new tasks to your list.
- **View Tasks**: View all your current tasks.
- **Delete Tasks**: Delete tasks you no longer need.
- **Easy Navigation**: Use a simple menu interface.

## How to Run in Visual Studio Code

1. Make sure you have Python 3.x installed on your machine.
2. Open the project folder in Visual Studio Code.
3. Open the integrated terminal in VS Code (you can do this by going to the Terminal menu and selecting `New Terminal`).
4. In the terminal, run the command `python todo_cli.py`.
5. Follow the on-screen instructions in the terminal to add, view, or delete tasks.

## Dependencies

- Python 3.x (no additional libraries needed)
